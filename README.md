# MAISON — E-commerce de Roupas Premium

> Stack: HTML · TailwindCSS · JavaScript · Node.js · Express · MySQL

-O Maison é um sistema de e-commerce de roupas premium que simula uma loja virtual completa, proporcionando uma experiência real de compra online. A aplicação permite que o usuário navegue pelos produtos disponíveis, utilize filtros de busca, visualize detalhes de cada item e adicione produtos ao carrinho de forma simples e intuitiva.

O processo de compra funciona de maneira fluida: o usuário escolhe os produtos, adiciona ao carrinho e segue para o checkout, onde pode revisar os itens antes de finalizar o pedido. O sistema mantém o controle do carrinho e garante uma navegação contínua, simulando um ambiente real de loja virtual.

A plataforma também conta com um sistema de autenticação, onde o usuário pode se cadastrar e realizar login. Após autenticado, ele passa a ter acesso a funcionalidades personalizadas, como acompanhamento de pedidos e gerenciamento de sua conta. Além disso, existe uma área administrativa restrita, onde é possível gerenciar produtos, pedidos e usuários, tornando o sistema completo tanto para clientes quanto para administradores.

O projeto foi desenvolvido com foco em usabilidade, organização de código e experiência do usuário, integrando frontend moderno com um backend estruturado e banco de dados, representando na prática o funcionamento de um e-commerce real.
